## Overview

This repository includes the head motion parameters figures for each subject.

Methods:
- data variance (dvars)
- framewise-displacement (FD)
- temporal signal-to-noise ratio (tSNR) 


## Directory Structure
```bash
behav/
    ├── D001/   # 'D' indicates dyscalculia
    │
    ├── .../   
    │
    ├── N001/   # 'N' indicates normal control
    │
    ├── README.md
```
